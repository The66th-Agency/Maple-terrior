## Brand Visual Identity

**Color palette:** Warm cream (#FDFBF7), amber gold (#C4841D), deep espresso (#1A1714), warm grays
**Mood:** Premium editorial, warm, organic, Canadian wilderness meets refined studio photography
**Lighting:** Soft natural light, golden hour warmth, no harsh shadows, no neon, no cool blue tones
**Texture:** Linen, raw wood, maple bark, parchment, brushed brass
**Avoid:** Plastic look, glossy reflections, blue/purple tones, sterile white backgrounds, stock photo feel, oversaturated colors

## How to Use with Reference Images

### Midjourney:
1. Upload the existing product image in the chat
2. Paste the prompt template
3. Add `--cref [uploaded image URL]` for character/style reference
4. Or drag the image into the prompt for image-to-image

### DALL-E / ChatGPT:
1. Upload the existing product image
2. Say: "Using this product as reference, generate: [paste prompt]"

### Ideogram / Flux:
1. Upload reference image
2. Paste prompt
3. Set style to "Realistic" or "Photo"
